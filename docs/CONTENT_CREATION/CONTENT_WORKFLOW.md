# 内容创建工作流程规范

## 概述

本文档说明 AI 如何从零开始准备和创建一个新的内容文件，包括数据收集、翻译、内容撰写等完整流程。

## 完整工作流程

### 阶段 1：选题与数据收集

收集论文、代码或资源的相关信息。

**详细流程**：[数据收集指南](01_DATA_COLLECTION.md)

### 阶段 2：翻译与本地化

将英文内容翻译为中文，确保术语一致性。

**详细流程**：[翻译与本地化指南](02_TRANSLATION.md)

### 阶段 3：内容撰写

按照规范撰写内容，确保结构清晰、内容准确。

**详细流程**：[内容撰写指南](03_CONTENT_WRITING.md)

### 阶段 4：分类标签管理

使用规范的分类标签，确保标签一致性。

**详细流程**：[分类标签管理指南](04_CATEGORY_MANAGEMENT.md)

### 阶段 5：内容审核

按照审核清单检查内容质量。

**详细流程**：[内容审核指南](05_CONTENT_REVIEW.md)

### 阶段 6：文件创建与提交

创建文件并提交代码。

**详细流程**：[文件创建与提交指南](06_FILE_CREATION.md)

### 核心原则

1. **数据真实性**：如果用户未提供数据，不要编撰，保持空缺
2. **术语一致性**：使用统一的术语翻译对照表
3. **内容简洁性**：代码文件不超过 100 行，MDX 文档不限制
4. **标签规范性**：使用已有标签，避免创建新标签
5. **质量可控性**：按照审核清单逐项检查
