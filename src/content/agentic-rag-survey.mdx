## 概述

Agentic-RAG explores advanced Retrieval-Augmented Generation systems enhanced with AI LLM agents.

## 核心内容

本文系统性地回顾了 Agentic RAG 领域的研究进展，探讨了如何通过 Agent 增强传统的 RAG 系统。

### 主要贡献

1. 系统性梳理了 Agentic RAG 的研究脉络
2. 提出了 Agentic RAG 的分类框架
3. 分析了不同方法的优缺点
4. 展望了未来的研究方向

## 关键特性

### 反思机制
Agent 能够对检索和生成过程进行反思，持续优化结果。

### 工具使用
通过调用外部工具增强检索和生成能力。

### 多 Agent 协作
多个 Agent 协同工作，分工处理不同任务。
